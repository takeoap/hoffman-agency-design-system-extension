01 / Rigged walk cycle

Make the complex walk.

One SVG figure, six joints. Each limb angle is a sine wave offset in phase — that's a walk. No frames, no video: the pose is recomputed every tick from time.
02 / Idle character

Even a cat is just code.

Shapes + three loops. The body scales a hair to "breathe," the eyes squash to blink on a timer, the tail sways on a sine. Same maths as the walk — different rig.
03 / Keyed action

Give the win a jump.

A keyframed leap. Vertical position follows an ease curve while the limbs tuck and extend — a parabola plus four angles. This is exactly how hand-drawn animation works, written as functions.
