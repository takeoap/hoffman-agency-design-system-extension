Fluent emoji

Microsoft Fluent emoji — our house set for visual storytelling. Drop one in with <fluent-emoji name="rocket"></fluent-emoji>. Add variant="animated" for motion, or size="120" to use it as a graphic. Curated set below — click any tile to copy its tag.

How it works. The <fluent-emoji> element resolves a local copy first (the curated set is in assets/emoji/, so it works offline) and falls back to the jsDelivr CDN for anything not localised — the full Fluent library is reachable by name. A bad animated path degrades gracefully to the colour version. Static art is from microsoft/fluentui-emoji; animated from Tarikul-Islam-Anik/Animated-Fluent-Emojis (both MIT). Decks built from the deck/keynote templates load the injector automatically.
