Media placeholders

Leave a labelled hole, not an empty one.

When a layout needs imagery the AI can't place, drop a placeholder that art-directs the shot. A person swaps in the real asset later. Three lines do the work: __label (what it is), __hint (art direction for the human), __prompt (a paste-ready seed for an image generator). Never a bare grey box.

How to use
  1. .tha-placeholder + a ratio modifier: --16x9, --4x3, --1x1, --4x5, --3x1, or --icon.
  2. Drop in the icon <svg class="tha-placeholder__icon">, then a .tha-placeholder__label formatted Type · Aspect · generate W×Hpx — the recommended pixel resolution is required, never the aspect alone, so whoever generates the image knows the exact size (16:9 → 1920×1080 · 4:3 → 1440×1080 · 3:4 → 1080×1440 · 4:5 → 1080×1350 · 1:1 → 1280×1280 · 3:1 → 1800×600 · 2:1 → 1600×800 · 9:16 → 1080×1920). Then a .tha-placeholder__hint (art direction for the human) and a .tha-placeholder__prompt (one or two sentences a generator can run with — bake in the imagery direction: candid, natural light, warm grade).
  3. .on-dark for navy slides · --accent for a solid lime hero frame.
  4. Optional: add data-slot="<unique-id>" and the brand-deck script makes the frame drop-to-fill — drag a real image on, it persists; hover → ✕ to clear. Prompts become click-to-copy. See slides/Hoffman Brand Guidelines.html (slides 30b–30f).
NEVER · ship a deck with an unlabelled grey box · stretch a photo to fill the wrong ratio · use a placeholder as a final asset
